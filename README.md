Mini RPG com Árvore Binária de Busca

Este projeto é um Mini RPG desenvolvido em Python, utilizando uma Árvore Binária de Busca (BST) para organizar os itens do inventário do jogo.

 Sobre o projeto

A ideia do jogo é simular um sistema de inventário onde o jogador pode:

- Explorar e encontrar itens aleatórios
- Adicionar itens ao inventário
- Remover itens
- Visualizar os itens organizados automaticamente

A organização é feita com base na **raridade do item**, usando uma árvore binária.



 Estrutura usada

O projeto utiliza:

- Classe Item (nome, tipo e raridade)
- Classe No (estrutura da árvore)
- Classe ArvoreBST (lógica da árvore binária)



Funcionalidades

- Geração de itens aleatórios
- Inserção automática na árvore
- Remoção de itens
- Busca de itens
- Listagem em ordem (do menor para o maior valor de raridade)
